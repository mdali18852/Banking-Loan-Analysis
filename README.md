## End-to-End Data Analytics Project using Excel, MySQL & Power BI

## Project Overview
This project presents an end-to-end Bank Loan Analytics Dashboard developed using Excel, MySQL, and Power BI. The objective is to transform raw loan data into meaningful business insights through data cleaning, SQL-based analysis, and interactive dashboards. The dashboard helps monitor loan performance, customer risk, funding trends, and repayment behavior to support data-driven lending decisions.

## Business Problem
Financial institutions need to monitor loan performance, identify high-risk loans, and evaluate repayment trends to make informed lending decisions.

This project helps answer questions such as:

What is the overall loan portfolio performance?
How do loan applications and funded amounts change over time?
What percentage of loans are Good vs Bad?
Which states and loan purposes contribute the most?
How can business decisions be improved using data?

## Key Business Insights

* Total Loan Applications reached 38.6K.
* Total Funded Amount exceeded $435.8M.
* Good Loans account for 86.2% of the total portfolio.
* Bad Loans represent 13.8%, requiring closer monitoring.
* Debt Consolidation is the most common loan purpose.
* Mortgage customers contribute a significant portion of funded loans.
* Loan applications show an increasing trend toward the end of the year.
