# Banking Customer Analytics Dashboard

## Project Overview
This project analyzes banking customer data to identify customer segments, loan exposure, deposit trends, and portfolio performance.

## Tools Used
- Excel
- MySQL
- Power BI
- DAX
- Power Query

## Business Objectives
- Identify high-value customers
- Analyze loan and deposit trends
- Monitor customer risk profiles
- Generate business recommendations

## Key Insights
- Total Deposits: ₹1.97Bn
- Total Loans: ₹1.74Bn
- High-value customers contribute significantly to deposits.
- Adult customers hold the highest loan exposure.
